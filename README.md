**팀원**

- 조계원

### 설치 및 실행 방법

```
// 프로젝트 Clone 방법
git clone https://github.com/LG-Uplus-Movie-SNS-PICKY/LG-Uplus-Movie-SNS-PICKY-FE.git
cd repository
```

```
// 의존성 설치
yarn intall

// 패키지 추가
yarn add { 패키지명 }
```

```
// 로컬 서버 실행
yarn dev
```
